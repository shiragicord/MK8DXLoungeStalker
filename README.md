# MK8DXLoungeStalker

## Requirement
- [uv](https://github.com/astral-sh/uv)

## Installation
Build Project enviroment with uv.
```bash
uv sync
```

## Usage
Run `main.py` with uv
```bash
uv run src/main.py
```
